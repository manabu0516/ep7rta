## ■ はじめに

Smilegateが開発と運営を行っているゲームのエピックセブンに関する非公式Discord Botの管理リポジトリになります。<br/>
※Twitterの有料化と公式がゲーム内に機能を実装したのを理由に更新止めてましたが、作りたい機能出てきたので再開しました。

Discord BOTを導入するとエピックセブンに関する以下のデータをDiscord上で表示することができます。

- RTAにおける対象編成を入力するとその編成に過去勝利した編成を一覧で表示します。
- RTAにおける対象編成を入力するとその編成に過去勝利した編成にもっとも含まれている英雄を一覧で表示します。
- RTAにおけるPick状態を入力すると、次によくPickされている英雄を一覧で表示します。
- 入力に使用する英雄のコード値を出力します。

## ■ インストールURL

https://discord.com/api/oauth2/authorize?client_id=1035058107899981854&permissions=395137001472&scope=bot

## ■ 基本的な使い方

インストールするとスラッシュコマンドとしてDiscordサーバに登録されます。

```
/ep7-rta-battle
/ep7-rta-battle2
/ep7-rta-pic
/ep7-rta-code
```

詳細な利用方法は実際にコマンドは以下URLより確認してください。
https://manabu0516.github.io/ep7rta/bot.html
